# Definición

> Es un potente editor de secuencias de texto. Puede hacer inserción, eliminación, búsqueda y reemplazo.

## Comando

```bash
sed 's/^ *//'
```
- Para quitar los espacios al principio. (^) empiece por espacio en este caso.
